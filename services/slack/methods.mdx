## Chat Methods

### chat.postMessage

Post a message to a channel.

```bash
POST /chat.postMessage
```

```json
{
  "channel": "C01GENERAL99",
  "text": "Hello World!",
  "thread_ts": "1234567890.000100",  // Optional: reply to thread
  "attachments": [...],  // Optional
  "blocks": [...]  // Optional
}
```

**Response:**
```json
{
  "ok": true,
  "channel": "C01GENERAL99",
  "ts": "1732645891.000200",
  "message": {
    "type": "message",
    "user": "U01AGENBOT9",
    "text": "Hello World!",
    "ts": "1732645891.000200"
  }
}
```

### chat.update


## Conversations Methods

### conversations.list

List all channels the user can see.

```bash
GET /conversations.list
```

**Query Parameters:**
- `types` - Channel types: `public_channel`, `private_channel`, `mpim`, `im`
- `limit` - Results per page (default: 100)
- `cursor` - Pagination cursor

**Response:**
```json
{
  "ok": true,
  "channels": [
    {
      "id": "C01GENERAL99",
      "name": "general",
      "is_channel": true,
      "is_private": false,
      "num_members": 3
    }
  ],
  "response_metadata": {
    "next_cursor": ""
  }
}
```

### conversations.info

Get information about a channel.

```bash
GET /conversations.info?channel=C01GENERAL99
```

### conversations.history

Get messages from a channel.

```bash
GET /conversations.history?channel=C01GENERAL99
```

**Query Parameters:**
- `channel` - Channel ID (required)
- `limit` - Results per page
- `oldest` - Start of time range
- `latest` - End of time range

### conversations.replies

Get thread replies.

```bash
GET /conversations.replies?channel=C01GENERAL99&ts=1234567890.000100
```

### conversations.open

Open a DM or group DM.

```bash
POST /conversations.open
```

```json
{
  "users": "U02JOHNDOE1,U03JANEDOE2"
}
```

### conversations.create

Create a new channel.

```bash
POST /conversations.create
```

```json
{
  "name": "my-new-channel",
  "is_private": false
}
```

### conversations.invite

Invite users to a channel.

```bash

```

```json

```

### conversations.leave

## Users Methods

### users.info

Get information about a user.

```bash
GET /users.info?user=U01AGENBOT9
```

**Response:**
```json
{
  "ok": true,
  "user": {
    "id": "U01AGENBOT9",
    "name": "agent_bot",
    "real_name": "Agent Bot",
    "profile": {
      "display_name": "Agent Bot",
      "email": "agent@example.com"
    }
  }
}
```

### users.list

List all users.

```bash
GET /users.list
```

### users.lookupByEmail

Find a user by email.

```bash
GET /users.lookupByEmail?email=john@example.com
```
